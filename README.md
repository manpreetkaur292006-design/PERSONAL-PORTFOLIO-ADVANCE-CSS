PERSONAL-PORTFOLIO-ADVANCE-CSS

This is a project on personal portfolio website using HTML and Advance CSS.

In this portfolio website, I have created the basic structure by using HTML and given style to the website using the Advance CSS.
This website includes alots of minor details.

This website includes a navigation bar on the top on which my name is written on the left corner and all the navigation options are written on the right side.
They are evenly spaced and I have added the hover and active effect to these options on hovering and clicking these options the text color, background color and the size changes.

The backgournd of this website inclues a beautiful gradient effect.

Then the home section have the welcome message whose colour is changing that i have done using the keyframes animation, this is placed on the left side of the site and my profile picture is added on the right side of this portion
that i have done using the display:flex; property. The profile picture is also having the box-shadow property that is adding a beautiful shadow effect.

Then in the about section you can see my Intro written in a box having the box-shadow effect and different colour then that of the backgrond along with the round borders. 

After this you can see the technical skills section, in this section i have included all my technical skills in the boxes that i have created with div tag and aligned all these boxes by using grid property,
along with a beautiful hover effect.

After this you can see My projects sections, in which i have listed all my best projects that i have made till now.
I have added the css keyframes animations to the image that i have pasted in that which shows a slideshow effect and when you click on the images you will reach to that site on which i have posted this projects
, either on the my linkedin profile or on my github repo. 
Each these projects are enclosed in separate boxes having the box-shadow effect along with a brief description of my project.

After this section you can see the certificates section, In which i have added all my certificates that i have gained till now along with a brief description about it.
All these are also enclosed in the boxes having the box-shadow effect and these boxes are placed side ny side that i have done using the display:flex; property of css.

After this you can see the contact me section, In which you can see a form that takes the input of the name, E-mail and the feedback all these have a very beautiful focus effect.

In the send button, you can see a beautiful hover effect with a gradient property.

Then you can see the social media section, in which i have added all my social media links that open when you click on their respective icons and these icons also have a hover effect.

Then you can see a back to top button that also have a hover and an active effect same as the navigation options.

In the end you can see the Thank you message that also have the css keyframe animation same as the welcome message.
Overall, i have made this using only HTML and Advance CSS.
